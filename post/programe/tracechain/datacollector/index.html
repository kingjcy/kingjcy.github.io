项目系列---- DataTransfer

数据分发程序是用于大量数据在不同组件之间传输，利用golang的高并发能力来实现，可以有助于数据转移，机器压力转移，支持tcp，http，grpc等方式的接入，支持kafka，file等方式的输出。具有资源利用少，效率高，速度快的特性。还支持etcd服务发现注册等功能。

tcp

http

grpc

ouput

kafka

file

整体

100--3000---20W/m---3000/s
