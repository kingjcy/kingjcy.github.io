Mongodb

mongodb是一种document类型的数据库，document我觉得事实上不要翻译的好，可以说是以一种包含key-value键值对的数据结构体，类似于json的格式bson。是一种面向集合的存储模式。我觉得集合和document在mangodb中很重要。mongo本身具有高性能，易部署等特性。

mongodb基本概念

这边最重要的就是集合，一种document类型的数据结构,如下举个例子就可以很形象的说明

{
    "_id" : ObjectId("54c955492b7c8eb21818bd09"),
    "address" : {
        "street" : "2 Avenue",
        "zipcode" : "10075",
    },
    "borough" : "Manhattan",
    "grades" : [
        {
            "date" : ISODate("2014-10-01T00:00:00Z"),
            "grade" : "A",
            "score" : 11
        }
        {
            "date" : ISODate("2014-10-01T00:00:00Z"),
            "grade" : "B",
            "score" : 17
        }
    ]
}

其中_id是唯一的标志。

安装与使用

我以centos 7为例，来安装mongodb。

  1. 我们可以用yum来安装，先配置yum源来加载包：

    vi /etc/yum.repos.d/mongodb-org-3.4.repo

    [mongodb-org-3.4] name=MongoDB Repository baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/3.4/x86_64/ gpgcheck=1 enabled=1 gpgkey=https://www.mongodb.org/static/pgp/server-3.4.asc

这个是3.0后的版本yum源，3.0之前的可以去官网查找，然后直接用yum命令安装

sudo yum install -y mongodb-org

  1. 可以用压缩包安装，直接去官网下载解压就可以用，这边就不赘述了。

  2. 相关路径

数据文件路径： /var/lib/mongo,但是默认是存储在/data/db这个目录下面，这个需要手动创建的，可以用–dbpath来改变。

日志路径： /var/log/mongodb

配置文件：/etc/mongod.conf

  1. 服务端

启动mongod服务：

systemctl start mongod

或者

/usr/bin/mongod -f /etc/mongod.conf

这边可以修改一下开机启动

sudo chkconfig mongod on

停止服务

systemctl stop mongod

重启服务

systemctl restart mongod
  1. 客户端

客户端是一个mongo的可执行程序，直接输入mongo就可以进入mongodb的shell界面。

具体的操作可以通过输入help进行查询。

  1. mongo基本命令的使用
  • dbname.system.* 是包含多种系统信息的特殊集合(Collection)
  • show dbs 展示数据库
  • use dnname 进入某数据库,如果数据库不存在则创建它。 这边和mysql的区别是mysql必须以；结束语句。
  • db.dropDatabase() 删除数据库
  • db 展示当前数据库
  • db.COLLECTION_NAME.insert(document) 插入数据
  • db.collection.update(query,update,{upsert: boolean,multi: boolean,writeConcern: document}) 1.查询条件，2.更新内容一般跟随变量使用，3.不存在是否插入一条，默认false不插入，4.默认false只修改第一条，否则全部修改 5.抛错级别
  • db.collection.remove(query,{justOne: boolean,writeConcern: document}) 1.查询条件 2.删除一条或者全部 3.抛错级别
  • db.集合名.find() 查找这个集合，如果需要用易读的方式来查询就加pretty(), and直接输入两个参数，or则用操作符$or 可以进行匹配
  1. 操作符
  • (>) 大于 - $gt
  • (<) 小于 - $lt
  • (>=) 大于等于 - $gte
  • (<=) 小于等于 - $lte
  • $type:数字对应不同的数据类型，匹配一个类型的数据。
